## 1. 产品概述

### 1.1 产品名称
小沓-BI

### 1.2 产品目标
面向海运货代接单业务，提供从来源输入到工单/工作单流转、字段质量、效率成本的统一分析视图，支撑业务运营、流程优化与成本评估。

### 1.3 当前页面结构
页面按以下结构组织：
1. 顶部筛选与操作区
2. KPI 卡片区
3. 分析模块区
4. 明细表与详情弹窗

页面已取消中间独立 tab 条，改为通过 KPI 卡片切换对应模块。

## 2. 顶部全局控制

### 2.1 控制项顺序
当前顶部从左到右为：
1. 分析维度
2. 公司 / 部门 / 个人
3. 时间范围
4. 全局成本参数
5. 来源
6. 导出

### 2.2 分析维度
支持两种分析维度：
1. 工作单维度
2. 人效维度

规则：
1. 工作单维度展示转化分析、数据质量分析、效率成本分析。
2. 人效维度仅展示效率成本分析。
3. 分析维度会联动 KPI、图表、明细表和导出结构。

### 2.3 组织选择
当前版本已移除租户概念。

规则：
1. 工作单维度下，组织固定为“全公司”。
2. 人效维度下，支持选择“全公司 / 部门 / 个人”。
3. 组织选择影响全部 KPI、图表、明细表与导出。

### 2.4 时间范围
支持日期区间选择，内置快捷选项：
1. 最近7天
2. 最近1个月
3. 最近3个月
4. 最近一年

规则：
1. 时间范围为全局生效条件。
2. KPI 环比采用真环比，即与上一个相邻等长周期比较。
3. 图表、明细表、导出均按当前时间范围重新计算。

### 2.5 来源
支持以下来源：
1. 全部来源
2. 邮件接单
3. 文件接单

规则：
1. 来源切换作用于两种分析维度。
2. 全部来源下保留整体数据，不再做字段识别渠道拆分展示。
3. 切换来源后，KPI、图表、明细表、导出同步刷新。

### 2.6 全局成本参数
全局成本参数已收纳为顶部弹层配置，包含 4 项可编辑参数：
1. 客服成本
2. 操作成本
3. 原校对时长
4. 原审核时长

规则：
1. 参数对工作单维度与人效维度同时生效。
2. 参数影响节省成本、时长成本、平均人力成本及相关导出结果。

### 2.7 筛选记忆
刷新页面后自动恢复上一次的主要筛选状态。

当前记忆内容包括：
1. 分析维度
2. 时间范围
3. 组织路径
4. 来源
5. 当前模块

## 3. 工作单维度

### 3.1 KPI
当前保留 3 张 KPI 卡片：
1. 工作单提交量
2. 字段一次通过率
3. 节省总成本

映射关系：
1. 工作单提交量 -> 转化分析
2. 字段一次通过率 -> 数据质量分析
3. 节省总成本 -> 效率成本分析

### 3.2 转化分析
转化分析当前包含两部分：
1. 业务单量趋势
2. 漏单深度分析

业务单量趋势展示 4 个阶段：
1. 来源输入
2. 成功创建工单
3. 工作单
4. 成功提交委托

漏单深度分析展示：
1. 当前漏单率
2. 漏单原因占比

当前漏单原因保留：
1. 接口超时
2. 文件解析失败

### 3.3 数据质量分析
数据质量分析包含两部分：
1. 整体质量雷达图
2. 字段识别准确率（字段维度）

整体质量雷达图当前展示 6 个指标：
1. 文件识别准确率
2. 邮件识别准确率
3. 字段一次通过率
4. 字段未修改率
5. 字段无需补录率
6. 字段保留率

#### 3.3.1 字段识别准确率（字段维度）
表头：
1. 字段名称
2. 识别准确率

展示规则：
1. 百分比展示
2. 保留 1 位小数
3. 支持按准确率排序
4. 默认按识别准确率倒序

统计口径：
1. 目标口径为 `field_correct_cnt / field_total_cnt`
2. 含义为“初始值等于提交值的工作单数 / 总工作单数”

### 3.4 效率成本分析
工作单维度下，该模块用于观察整体处理效率与成本。

当前图表包括：
1. 节省成本与单量趋势
2. 平均处理时长趋势
3. 工作单提交与返工分布

### 3.5 工作单维度明细表
根据当前模块切换不同明细结构。

#### 3.5.1 转化明细
字段：
1. 来源ID
2. 工单ID
3. 接单来源
4. 状态
5. 漏单原因

#### 3.5.2 质量明细

7. 字段未修改率
8. 字段无需补录率
9. 字段保留率
10. 跟进人
11. 结束时间

详情弹窗标题：字段识别及编辑情况

详情弹窗字段：
1. 字段

## 5. 导出

### 5.1 导出类型
根据当前分析维度导出：
1. 工作单分析
2. 人效分析

### 5.2 文件命名
当前命名规则为：

```text
工作单分析-来源-组织-时间.xlsx
人效分析-来源-组织-时间.xlsx
```

### 5.3 工作单维度导出

1. 转化分析
2. 数据质量分析

字段识别准确率（字段维度）表头为：
1. 字段名称
2. 识别准确率

4. 节省总成本
